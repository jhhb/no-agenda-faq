# No Agenda FAQ

An FAQ for the [No Agenda Show](noagendashow.net), the best podcast in the universe.

This project is built using Next.js.

# Local development

## Dependencies

```
- nodejs
- yarn
```

## Running locally

```bash
cd no-agenda-faq/
yarn && yarn dev
```
